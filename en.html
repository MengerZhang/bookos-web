What isBook OS? Book is the ladder of human progress - golgi. so I call itBook OS。
Book OSis a x86 platform 32 bits Operating System.
Support
- i386 platform
- ELF format kernel and program
- paging memory management
- multiple processes and kernel threads
- process synchronization and queue
- node physical memory management
- virtual memory management
- advanced interrupt mechanism
- block devices and character device frames
- IDE hard drive
- memory disk (Ramdisk)
- BOFS file system
- timer
- keyboard driver
- mouse driven
- user library
- graphics driver
- simple hardware abstraction driver
- tty drive
- signal system
- serial driver
- buzzer driven
- RTL8139 network card driver
- LIBC library file
- KGC graphics core
- KGC GUI
